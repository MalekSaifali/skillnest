# SkillNest

A skill-exchange platform for students and professionals.

## Services
- user-service    : Profile, privacy settings
- follow-service  : Follow / unfollow public profiles
- connect-service : Friend requests
- chat-service    : Real-time messaging between friends
- search-service  : Search profiles by skill

## Tech Stack
- Frontend : Next.js (React)
- Backend  : Node.js + Express
- Database : PostgreSQL
- Auth     : AWS Cognito
- Realtime : Socket.io
- Cache    : Redis
- Search   : Elasticsearch
- Storage  : AWS S3
